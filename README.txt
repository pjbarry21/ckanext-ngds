   NGDS: The National Geothermal Data System
===============================================

THIS IS A PRELIMINARY DOCUMENT!

NGDS provides a solution for storing and retrieving information about 
geothermal energy. It provides a web user interface with flexible search and
analysis features. NGDS also provides all features required by an orginization
to maintain a repository for geothermal data including flexible bulk upload
mechanisms and a business process that allows for quality assurance from the
repository owner.
NGDS is based on CKAN, one of the most frequently used open source data 
portals. NGDS is flexible and can be tailored and adjusted to specific needs.

Installation
============

See the NGDS Documentation for installation instructions.

Support
=======

If you need help with NGDS or want to ask a question about NGDS, use the 
ngds-discuss mailing list (try searching the ngds-discuss archives for an
answer to your question first).

Contributing to NGDS
====================

For contributing to NGDS or its documentation, see CONTRIBUTING.

If you've figured out how to do something with NGDS and want to document it
for others, make a new page on the NGDS wiki, and tell us about it on 
ngds-discuss.
